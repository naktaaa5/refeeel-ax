# RE:FEEEL AX — 대시보드

1인 브랜드 운영 워크스페이스(AX) 대시보드. 옵시디언 볼트를 브라우저에서 직접 읽어 한 화면에 모읍니다.

## 쓰는 법 (Mac · Windows 공통)

1. **크롬 또는 엣지**로 이 페이지 열기 (Firefox·Safari는 폴더 열기 미지원)
2. 좌하단 **볼트 열기** → 옵시디언 볼트 루트 폴더(`낙타`) 선택
3. 한 번 고르면 그 PC 브라우저에 기억됨 — 다음부턴 버튼 한 번이면 재연결

## 데이터는 어디에?

- **볼트 내용은 어디에도 업로드되지 않습니다.** 브라우저가 로컬 폴더를 직접 읽을 뿐(File System Access API).
- 이 저장소엔 대시보드 **코드(HTML)만** 있습니다. 컨설팅·릴스·매출 등 실제 데이터는 각자 PC 볼트에만 존재합니다.
- 그래서 Mac에서 만든 페이지를 Windows에서 열어도, Windows의 볼트를 고르면 그 PC 데이터를 봅니다.

## 업데이트(재배포)

대시보드 원본(`통합워크스페이스/dashboard.html`)을 고친 뒤:

```bash
./deploy.sh
```

→ `index.html`로 복사 후 푸시. 1~2분 뒤 반영됩니다.
